# Controller

## Filtre

- creation de filtre sur la table courant
- ajout des modification dans un historique de table

## Converter

- creation de "Dataset"/"DataGroup" a partir des donnes CSV
- summary des donnees (min, max, moyenne, etc) pour les afficher dans la table de droite

## Chart

- utilisation des dataset pour creer des graphiques

## Loader

- chargement de fichiers CSV

## Report

- creation de rapport en String a partir des dataset et des graphiques
- generation de summary + combiner avec graphique
