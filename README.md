## Setup

### Logiciels nécessaires

- **VS Code**
- **Git**
- **Java JDK 21**

## Installation

### Cloner le dépôt

```bash
git clone https://github.com/jules1univ/VectorReport.git
cd VectorReport
```

### Ouvrir le projet

- Ouvrir le dossier `VectorReport` dans **VS Code**
- Vérifier que le JDK est bien configuré (`java --version`)

## Libraries

- **DuckDB JDBC 1.2.1**

  > Embedded analytical database used to efficiently query CSV data.

- **FlatLaf 3.7.1**

  > Modern Look & Feel for the graphical user interface.

- **FlatLaf Extras 3.7.1**

  > Additional UI components and utilities for FlatLaf.

- **SVG Salamander 1.1.5.3**

  > Library used to render and display SVG graphics generated by the application.

- **JUnit 4.13.2**
  > Framework used for unit testing.

## Lancer l'application

Après avoir cloné et ouvert le projet :

```bash
javac -cp "lib/*" application/VectorReport.java
```

Ou, si un JAR est construit via le workflow :

```bash
java -jar VectorReport.jar
```
