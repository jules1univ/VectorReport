## Setup

### Logiciels nécessaires

- **VS Code**
- **Git**
- **Java JDK 21**

## Installation

### Cloner le dépôt

```bash
git clone https://github.com/jules1univ/VectorReport.git
cd VectorReport
```

### Ouvrir le projet

- Ouvrir le dossier `VectorReport` dans **VS Code**
- Vérifier que le JDK est bien configuré (`java --version`)

### Librairies

- JUnit 4.13.2

  > Utilisé pour les tests unitaires.

- FlatLaf 3.7.1

  > Look and Feel moderne pour l'interface graphique.

- SVG Salamander 1.1.5.3
  > Librairie de rendu SVG utilisée pour afficher les graphiques générés.
